### TaskMaster - Task Tracking & Management System
TaskMaster is a robust backend system for managing tasks and projects, designed to facilitate team collaboration. Built with Java and Spring Boot, it features secure authentication, real-time data management, and GenAI integration using Google Gemini to automatically assist with task creation.

Core Functionality
User Management: Secure Registration, Login (JWT-based), and Profile updates.

**Project Management**: Create projects, manage descriptions, and invite team members.

**Task Management**: Full CRUD operations for tasks with titles, descriptions, due dates, and priority status.

**Collaboration**: Assign tasks to specific users, add comments, and attach external resources (URLs).

**Search & Filter**: Advanced filtering by task status and keyword search for task titles/descriptions.

My Tasks: Dedicated view for users to see all tasks assigned specifically to them across different projects.

Smart Descriptions: Integrated with Google Gemini AI to automatically generate professional task descriptions based on simple titles.
